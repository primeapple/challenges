# Cassidoo

The blogger "Cassidoo Wiliams" posts a newsletter every week. Each time she includes a "Interview Question", which is a very simple software kata.
You can find them here: https://buttondown.com/cassidoo/archive

I try to solve them every week to learn a little more Go.
I try to do it myself first, in a TDD style.
After that, I let the AI create a solution and ask it to compare them.

## Usage

Run all tests 
```sh
go test ./...
```
