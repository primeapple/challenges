This week's question:
Write a function that determines if a number is abundant, deficient, perfect, or amicable. (Thanks Raymond for this one!) 
See https://www.encyclopedia.com/education/news-wires-white-papers-and-books/numbers-abundant-deficient-perfect-and-amicable

Example:

```js
whatKindOfNumber(6)
> 'perfect'

whatKindOfNumber(12)
> 'abundant'

whatKindOfNumber(4)
> 'deficient'
```
